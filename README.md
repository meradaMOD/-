# المنصة التعليمية الجزائرية

موقع تعليمي مجاني يهدف إلى توفير محتوى تعليمي عالي الجودة للطلاب الجزائريين.

## المميزات
- دروس تفاعلية
- محتوى فيديو تعليمي
- واجهة سهلة الاستخدام
- تصميم متجاوب مع جميع الأجهزة

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript
- Font Awesome
- AOS (Animate On Scroll)
